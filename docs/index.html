06

Método y principios

Solo datos públicos. Todo proviene de las APIs de datos abiertos que el propio Estado colombiano publica: SECOP II vía datos.gov.co, de la Agencia Nacional de Contratación Pública, Colombia Compra Eficiente.

Metodología auditable. Cada bandera es una consulta SQL visible en el repositorio. Cualquier persona puede reproducir cada número de esta página ejecutando tres comandos.

Inspiración. Este proyecto industrializa la metodología de veeduría ciudadana que consiste en revisar sistemáticamente el SECOP y cruzarlo con contexto público.

Una bandera roja es una hipótesis, no un veredicto. El sistema señala dónde mirar; juzgar corresponde a los ciudadanos, a los periodistas y a los órganos de control.

La transparencia es un bien público.

El código, la taxonomía de 27 patrones de irregularidad y el diseño completo del sistema son abiertos.
